# Claude Code 활용 메모

이 프로젝트를 작업하며 사용한 Claude 관련 기능을 간단히 정리한 문서입니다.

## Claude Code

터미널·IDE·웹에서 동작하는 Anthropic의 공식 코딩 에이전트. 코드베이스를 이해하고,
파일을 직접 수정하고, Git 작업까지 수행할 수 있다.

## Artifact (비포/애프터 시각화)

- 자체 완결형 HTML 페이지를 만들어 링크로 바로 확인할 수 있는 기능.
- 코드 변경 결과 비교, UI 목업, 데이터 시각화, 문서/리포트 등에 활용.
- 별도 명령어 없이 "before/after로 보여줘", "비교 페이지로 만들어줘"처럼 자연어로 요청.
- 예: 레포 카드 높이 수정 전/후를 실제 픽셀 값과 함께 나란히 비교.

## 전역 설정 (`~/.claude/CLAUDE.md`)

- 홈 디렉토리의 `~/.claude/CLAUDE.md`에 적은 지침은 **모든 프로젝트·모든 세션**에 적용된다.
- 특정 프로젝트에만 적용하려면 프로젝트 루트의 `CLAUDE.md`를 사용.

### Finder에서 숨겨진 `.claude` 폴더 보기

`.claude`는 이름이 점(`.`)으로 시작해 Finder에서 기본적으로 숨겨져 있다. 세 가지 방법:

1. **명령어로 바로 열기** — 터미널에서 `open -R ~/.claude/CLAUDE.md`
   실행 시 Finder가 해당 파일을 선택한 채로 열린다.
2. **Finder에서 경로로 이동** — `Cmd + Shift + G`(Go to Folder) →
   `~/.claude/` 입력 후 Enter.
3. **숨김 파일 표시 토글** — 홈 폴더(`Cmd + Shift + H`)에서
   `Cmd + Shift + .`(점)을 누르면 숨김 파일/폴더가 표시된다. 다시 누르면 숨김.

## Git 커밋 규칙

- 커밋은 **의미 단위로 쪼갠다.** 서로 다른 관심사를 한 커밋에 뭉치지 않는다.
- 커밋 메시지는 Conventional Commits 형식(`type(scope): 요약`)을 따른다.
- 커밋/푸시는 사용자가 요청할 때만 수행한다. (자동 커밋 아님)
